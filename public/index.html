Проекты:

  1. Блог с полным функционалом

    Разработан на основе технологий MERN (MongoDB, Express.js, React, Node.js). Проект включает в себя все необходимые функции для полноценной работы блога, такие как система аутентификации пользователей, возможность создания и редактирования постов, комментарии, а также адаптивный дизайн для различных устройств.

  2. Fullstack аналог кинопоиска

    Созданный с использованием принципов Fullstack разработки, этот проект предоставляет пользователям возможность поиска фильмов, просмотра их описаний, а также оставления отзывов. Для реализации использовались современные JavaScript библиотеки и фреймворки, что позволило создать высокопроизводительное и удобное приложение.

  3. Fullstack сервис подбора отелей

    В данном проекте была разработана система, позволяющая пользователям находить и бронировать отели по заданным критериям. Подбор отелей реализован с использованием API популярных сервисов, что обеспечило доступ к актуальной информации и снизило риски ошибок при показе данных.
